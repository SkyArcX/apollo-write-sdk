# Apollo Write SDK

> 让 Apollo 拥有 客户端写入配置的能力，像Diamond一样使用！

## 项目简介
**Apollo Write SDK**是一个基于Apollo Portal REST API封装的JAVA工具包，用于在客户端直接向Apollo配置中心**写入、修改、删除、发布**配置项

适用于：
- 想要在运行中的客户端动态修改配置；
- 需要通过代码进行批量配置管理；
- 希望在离线/自动化场景中操作Apollo；
- 希望Apollo拥有类似Diamond的能力
